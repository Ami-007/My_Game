This game follows sudoku rules. You may only have one of each icon in every row, column and 2 x 2 sqaure. Use the given icons to figure out the placement of the rest. To input an icon, click on the chosen button and then click the box you want it to apply to. Use the Reset Board button to reset the board back to the start. Use the check button to verify your results: - if a box turns red, the icon is incorrect, - if the box turns green, you're onto a win.
